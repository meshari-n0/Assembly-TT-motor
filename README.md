# Two-Wheel Robot Base (TT Motor)

A simple and lightweight **2-wheel robot chassis** designed for **TT motors**, suitable for robotics projects, education, and prototyping.  
The design is optimized for **3D printing** and easy assembly.

---

## Features
- Supports **TT DC Motors**
- Compatible with common **robot wheels**
- Lightweight and compact design
- Easy to assemble and modify
- Optimized for **FDM 3D Printing**
- Suitable for Arduino / Raspberry Pi projects

---

## Components
- 2 × TT Motors  
- 2 × Robot Wheels  
- 1 × Base Plate  
- Motor Holders / Brackets  

---

## Use Cases
- Line follower robot  
- Obstacle avoidance robot  
- Educational robotics kits  
- DIY robotics projects

---

## Assembly Notes
- Motors are fixed firmly to avoid vibration  
- Wheels rotate freely with motor shaft  
- Base allows future expansion (battery, controller, sensors)
